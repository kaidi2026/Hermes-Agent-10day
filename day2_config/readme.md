# Hermes Agent 实战训练营
# 本仓库提供训练营所需数据和代码

> 从零搭建属于你的自主AI助手，让AI真正跑在你的工作里

## 项目简介

Hermes Agent 是由 Nous Research 开发的开源 AI 智能体框架（MIT 许可证）。它不是一个简单的聊天机器人或编码工具，而是一个"住在你服务器上的自主 AI 助手"——越用越聪明。

与传统的 AI 工具不同，Hermes Agent 具备自主运行、自我学习和自我改进的能力。你可以把它部署在自己的服务器上，让它 24 小时不间断地为你工作。它会记住你的偏好、学习你的工作模式，并随着时间的推移变得越来越高效。

**一句话概括：Hermes 是第一个出厂就带"缰绳"的 Agent，而且缰绳会自己长大。**

## 核心特性

- 🚀 **24/7 自主运行**：部署后无需盯着，它会自己在后台持续工作
- 🔗 **多平台互联**：能通过钉钉、飞书、企业微信等平台随时联系，随时随地给它派活
- 🧠 **自我进化学习**：会从每次交互中学习，自动创建和改进技能（Skill），越用越聪明
- 🔓 **模型无关架构**：支持 200+ AI 模型，包括 OpenAI、Anthropic、Google、本地模型等，你可以自由选择

## Hermes vs OpenClaw

| 框架 | 哲学 | 核心价值 | 适用场景 |
|------|------|----------|----------|
| **OpenClaw** | "配置即行为"，你定义什么，Agent 就是什么 | 可预测、可审计、可复制 | 需要合规性和一致性的企业场景 |
| **Hermes Agent** | 自主后台引擎，自己记忆、自己学习、自己进化 | 自主性和自改进能力 | 个人效率提升、自动化工作流、长期助手 |

## 训练营目标

本训练营将带你从 0 开始，搭出自己的第一个 Hermes 助理，并基于其自身的记忆机制，逐步让它变得更理解你。

**记住：先把第一步走稳，比什么都重要。**

## 课程大纲

### 第一课 环境篇：从零运行，完成你的第一次部署
**目标**：带你把属于你的第一个 Hermes Agent 助理，真正配置好、跑起来

大多数人花 1 小时左右，就能完成第一轮基础搭建。有了这个基础，你就可以先尝试一些最直观的动作：
- 让它帮你写一段简单文案
- 让它帮你整理资料和信息
- 让它帮你做基础提醒或任务辅助
- 体验"AI 从回答问题，变成开始协助做事"的差别

---

### 第二课 配置篇：让 Hermes 在第一天就认识你
**目标**：让你的 Hermes 不只是能用，而是越来越像你想要的那个助手

解决默认配置下能力不足、不够稳定、不够适配个人需求的问题。

---

### 第三课 能力篇：安装 Skill，打造工具调用
**目标**：让 Hermes 真正参与到你的工作流程中

这一课我们将跑通一整条内容生产链路：
- 根据主题或素材生成文章初稿
- 按要求调整结构、语气和表达
- 补全标题、开头、正文和结尾
- 最终跑通自动发布流程

---

### 第四课 实战篇(上)：职场效率重构
**目标**：用 Hermes 搭建多平台内容生成系统，形成属于你自己的内容生产流水线

---

### 第五课 实战篇(下)：知识管理与研究提速
**目标**：利用 Hermes 构建个人知识库，大幅提升信息检索和研究效率

---

### 第六课 进阶篇：多 Agent 协同 + 定时任务 + 闭坑实录
**目标**：掌握高级功能，解决实际使用中的常见问题

---

### 第七课 方法论篇：如何让 Hermes 真正跑在你的工作里
**目标**：统一复盘总结，明确后续发展方向

通过直播形式回顾核心内容，梳理常见问题，帮助你找到最适合自己的深挖方向。
